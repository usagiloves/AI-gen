# Prompt Phase 1 - Video Analysis

Thư mục này chứa các file `.txt` prompt dùng cho Phase 1 (Phân tích Video).

## Cách sử dụng
- Đặt các file `.txt` vào đây, đặt tên theo thứ tự (01_, 02_, ...).
- Hệ thống sẽ đọc và gửi lần lượt theo thứ tự alphabet.
- File đầu tiên sẽ được gửi kèm với video đã upload.

## Lưu ý
- Hãy nạp file prompt `.txt` thực tế vào đây trước khi chạy pipeline.
