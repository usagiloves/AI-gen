# Prompt Phase 2 - Script Generation

Thư mục này chứa file template kịch bản cho Phase 2.

## File chính
- `Prompt KB.txt` — File cốt lõi chứa từ khóa `CÂU CHUYỆN:`. 
  Hệ thống sẽ tự inject nội dung tóm tắt từ Phase 1 vào sau dòng này.

## Lưu ý
- Hãy nạp file `Prompt KB.txt` thực tế vào đây trước khi chạy pipeline.
- Đảm bảo file chứa từ khóa `CÂU CHUYỆN:` (viết HOA, có dấu hai chấm).
